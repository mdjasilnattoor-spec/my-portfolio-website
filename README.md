# Portfolio Website

A personal portfolio website for **Muhammed Jasil**, Data Analyst.

## Features
- **Hero Section**: Introduction and resume download.
- **About Me**: Professional bio and education details.
- **Skills**: Categorized technical skills (Data Analysis, Visualization, ML, etc.).
- **Projects**: Showcase of GitHub repositories.
- **Contact**: Simple contact layout.
